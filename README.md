For Day 6 of my 30 Days, 30 Projects challenge, I developed a File Organizer Tool using Python and CustomTkinter to create a clean, modern desktop automation utility.

The application allows users to select any directory and automatically organizes files into categorized subfolders such as Images, Videos, Documents, Audio, Archives, Code, and Others. File categorization is handled through extension-based mapping, ensuring structured and predictable organization.

A key focus of this project was transparency and user feedback. Instead of simply moving files silently, the application generates a detailed organization report that includes:

Total number of files processed

Category-wise distribution summary

Individual file movement logs

Clear execution status updates

This reporting mechanism improves user confidence and makes the tool feel complete and production-ready rather than just a basic script.
